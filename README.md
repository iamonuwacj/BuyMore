# Internal Product Catalog

**Technologies:** React, REST API, Tailwind CSS

### Description
A responsive catalog interface for browsing products or services. Fetches data from a public API and includes category filtering and search.

### Features
- Display products with images, names, and prices
- Filter products by category
- Search products by name
- Responsive layout for all devices

### Installation
```bash
git clone <your-repo-url>
cd internal-product-catalog
npm install
npm start
```

### Usage
- Open the app at `http://localhost:3000`
- Use search and filter to locate products
- Click on product cards for details (if implemented)

### Notes
- Extended a public API starter template.
- Focused on clean layouts and reusable components.
